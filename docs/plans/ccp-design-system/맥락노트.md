# 맥락 노트: CCP 디자인 시스템

## 현재 코드베이스 상태

- pnpm 모노레포, FSD 아키텍처, React 19 + Vite 7 + Tailwind CSS 4
- `content/ccp/example` 파일에 Figma export 코드가 ~2000줄 모놀리식으로 존재
- `content/ccp/meta.ts`에 문서 메타데이터 정의됨
- `_components/` 디렉토리는 아직 없음 (신규 생성 예정)

## 핵심 파일

| 파일 경로 | 역할 | 변경 여부 |
|-----------|------|-----------|
| `apps/web/src/content/ccp/example` | Figma export 원본 (디자인 레퍼런스) | 참조 |
| `apps/web/src/content/ccp/meta.ts` | 문서 메타데이터 | 참조 |
| `apps/web/src/content/ccp/_components/` | CCP 디자인 시스템 (전체) | **신규** |
| `apps/web/src/shared/ui/components/` | Tino 기존 shared 컴포넌트 (text, diagram) | 참조 |
| `apps/web/src/shared/ui/layouts/AppLayout.tsx` | Tino 앱 레이아웃 | 참조 |
| `apps/web/package.json` | 웹앱 의존성 | 수정 (cva, clsx, tailwind-merge 추가 시) |

### 타이포그래피

#### 폰트 패밀리
```
Pretendard-Regular (400) → 본문, 보조 텍스트
Pretendard-Medium (500)  → 기본 라벨, 뱃지 텍스트
Pretendard-SemiBold (600) → 소제목, 강조
Pretendard-Bold (700)    → 제목, 주요 강조
```

#### 사이즈 스케일
```
10px → 아주 작은 라벨 (탭 아이콘 하단)
11px → 툴팁, 작은 주석
13px → 기본 본문 크기 (가장 많이 사용)
14px → sm 크기
16px → base 크기 (섹션 헤딩)
18px → lg 크기
20px → xl 크기 (페이지 제목)
24px → 2xl 크기 (큰 숫자)
```

#### Letter Spacing 패턴
```
fontSize px × -0.01 = letterSpacing
예: 13px → -0.13px, 14px → -0.14px, 20px → -0.20px
```

### 그림자
```
card:     0px 0px 8px #00000014    (가장 많이 사용)
dropdown: 4px 4px 4px #0000001a
tooltip:  4px 4px 8px #1b2c3f33
header:   0px 1px 5px #6b77ac33
panel:    -1px 0px 5px #6b77ac33
subtle:   2px 2px 8px #0000000d
focus:    0px 2px 10px #0077ff33
```

### Border Radius
```
sm:   4px  → 입력 필드, 작은 카드
md:   6px  → 기본 라운드
lg:   8px  → 카드, 드롭다운
pill: 18px~20px → 필 모양 버튼/뱃지
full: 50px → 원형 (상태 도트)
```

## 제약 조건

### 기술적 제약
- **FSD 규칙**: `_components/`는 content 내부이므로 FSD 레이어 규칙에서 자유로움
- **Tailwind v4**: `@theme` 디렉티브 사용 가능하지만, content 수준에서는 TypeScript 상수가 더 실용적
- **Pretendard 폰트**: 프로젝트에 이미 설치되어 있는지 확인 필요. 없으면 웹폰트 추가 필요
- **SVG 아이콘**: example 파일은 60+ SVG를 개별 import하지만, lucide-react로 대체 가능한 것은 대체

### 비즈니스 제약
- **디자인 충실도**: Figma 원본 디자인에 최대한 가깝게 구현
- **슬라이드 뷰**: 이 컴포넌트들은 Tino 슬라이드 안에서 렌더링됨 (1920×1050 뷰포트)
- **인터랙션**: 슬라이드 뷰어 모드에서는 클릭 등 인터랙션이 제한적이므로, 시각적 상태 표현에 집중

## 핵심 결정 로그

| 시점 | 결정 | 이유 | 대안 |
|------|------|------|------|
| 초기 | `_components/` 배치 | shared 오염 방지, co-location | shared/ui/ccp, packages/ccp-ui |
| 초기 | TS const 토큰 | 타입 안전, IDE 지원, 별도 CSS 불필요 | CSS custom properties, @theme |
| 초기 | cva + Tailwind | 타입 안전 variant, 기존 스택 호환 | styled-components, CSS modules |
| 초기 | lucide-react 아이콘 | 이미 설치됨, 깔끔한 API | SVG 직접 관리, @heroicons |

## 주의사항

- **example 파일의 절대 좌표**: `absolute top-[581px] left-[1795px]` 같은 절대 위치는 컴포넌트에서 제거하고, 레이아웃은 슬라이드 단에서 처리
- **Figma export 특이값**: `w-[-36471.43%]`, `h-[111271.43%]` 같은 비정상적 크기값은 무시 (SVG viewBox 계산 오류)
- **mt-[-1.00px] 패턴**: Figma가 텍스트 베이스라인 보정을 위해 삽입한 것. 컴포넌트에서는 Tailwind의 자연스러운 line-height로 대체
- **duplicate import**: example 파일에서 `useState`가 여러 번 import됨 (각 섹션이 원래 별도 파일이었기 때문)
- **폰트 이름**: `Pretendard-Bold`와 `Pretendard_GOV-Bold`가 혼용됨. GOV 버전은 정부 공공기관용 변형체
