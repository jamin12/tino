### 기술적 제약
- CCP는 Gitea + ArgoCD 기반으로, API 호출 시 두 시스템의 REST API를 조합해야 함
- 코드 에디터는 Monaco Editor(VS Code 기반) 사용
- 해상도: 1920x1050 기준 디자인

### 디자인 제약

- LNB 고정 260px, GNB 고정 60px
- 주요 색상: `#0077FF` (Primary Blue), `#1B2C3F` (Nav)
