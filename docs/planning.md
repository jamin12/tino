# Tino - 기능 목록

AI가 기획 문서를 자동 생성하고, 슬라이드/캔버스로 보여주는 도구.

---

## 핵심 흐름

프롬프트 입력 → AI가 문서 생성 → 슬라이드 or 캔버스로 뷰잉 → 직접 편집

---

## Phase 1 (MVP)

| 기능 | 설명 |
|------|------|
| AI 문서 생성 | 프롬프트를 입력하면 AI가 텍스트 + 다이어그램이 포함된 기획서를 자동 생성 |
| 슬라이드 뷰 | PPT처럼 슬라이드를 넘기며 보기. 썸네일 사이드바, 키보드 단축키 |
| 프레젠테이션 | 전체화면 발표 모드 |
| 문서 관리 | 문서 목록, 생성, 삭제, 자동 저장 |
| 텍스트 블록 | 제목, 본문, 리스트, 콜아웃, 코드블록, 테이블, 인용문 |
| 플로우차트 | 노드와 엣지로 흐름을 시각화. 자동 레이아웃 |
| 마인드맵 | 중심 주제에서 가지치기 형태의 아이디어 맵 |

## Phase 2

| 기능 | 설명 |
|------|------|
| 캔버스 뷰 | Figma처럼 무한 캔버스에 자유 배치, 줌/팬 |
| 인라인 편집 | 텍스트를 클릭해서 바로 수정 |
| 속성 패널 | 선택한 요소의 색상, 크기, 레이아웃 편집 |
| 시퀀스 다이어그램 | 시스템 간 메시지 흐름 시각화 |
| ER 다이어그램 | 데이터베이스 엔티티 관계도 |
| 아키텍처 도 | 시스템 구성 요소와 연결 관계 |
| 와이어프레임 | 레이아웃, 네비게이션, 폼, 카드, 모달, 모바일 목업 |
| AI 부분 편집 | 특정 슬라이드/요소만 AI에게 수정 요청 |
| Undo/Redo | 실행 취소/다시 실행 |
| 내보내기 | PDF, PNG, JSON |

## Phase 3

| 기능 | 설명 |
|------|------|
| 템플릿 | 자주 쓰는 문서 구조 재사용 |
| AI 멀티턴 대화 | AI와 대화하며 점진적으로 문서 완성 |
| 버전 관리 | 변경 이력 추적, 특정 버전 복원 |
| 클라우드 동기화 | 여러 기기에서 접근 |
| 이미지 분석 | 스크린샷 → AI가 와이어프레임으로 변환 |

---

## 컴포넌트 목록

### 텍스트 (7종)
Heading, Paragraph, BulletList, Callout, CodeBlock, Table, Quote

### 다이어그램 (7종)
Flowchart, SequenceDiagram, ERDiagram, MindMap, Architecture, Timeline, TreeDiagram

### 와이어프레임 (7종)
Layout, Nav, Form, Card, List, Modal, Mobile

### 구조 (6종)
Divider, Spacer, Grid, Columns, Image, Embed

## 화면 구성

| 경로 | 화면 |
|------|------|
| `/` | 문서 목록 (카드 그리드) |
| `/new` | 새 문서 (AI 프롬프트 입력) |
| `/document/:id` | 문서 뷰/편집 |
| `/document/:id/present` | 프레젠테이션 모드 |
